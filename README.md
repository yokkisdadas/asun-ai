# asun-ai
Asun AI, CrewAI ve Gemini altyapısıyla çalışan, insan müdahalesi olmadan 7/24 interneti tarayan otonom bir sistemdir. GitHub Actions sayesinde dünyadaki teknolojik boşlukları bulur ve kendi kaynak kodunu (main.py) yeniden yazarak evrimleşir. Supabase ve Vercel ile sıfır maliyetle verileri saklayıp dünyaya açılan, canlı bir AGI projesidir.
